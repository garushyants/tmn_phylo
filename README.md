This repository contains customary code for Wu, Zhang, Garushyants et al "Tmn blocks phage spread via plasmolysis and triggers synergistic defence responses", 2026

Folder *scripts* contains custom R and python scripts used for protein family characterization, phylogenetic analysis, investigation of genomic contexts & RNAseq analysis. 
Folder *figures* contains visualizations that were used as illustraition in the manuscript.
In addition to the dat provided in the paper in *figures/genomic_contexts* there are vizualization of genomic contexts for each leaf on the tree. The same information is provided in paper as a Supplementary table 2.

In *data* folder alignment and phylogenetic tree is provided as well as additional data to reproduce the analysis. Moreover, *data/pdbs* folder contains AlphaFold3 models for representatives of all clades that were used is structural comparisons of the models.
